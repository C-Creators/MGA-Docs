## Enseñando Nuevas Habilidades a Tu Asistente <Badge text="Extensible" variant="note" size="small" />

Los plugins son como cursos de entrenamiento especializados para tu asistente. Añaden nuevas habilidades que tu asistente no podría hacer de otra manera – como programar citas, enviar emails o procesar pagos. Al igual que enviar a un empleado a un curso de capacitación, los plugins expanden lo que tu asistente puede hacer por los clientes.

![Plugins disponibles](../../../../assets/screenshots/assistants-plugins.png)

<Aside type="tip" title="Eligiendo Habilidades">
- **Solo añade habilidades que tu asistente realmente necesite** – demasiadas pueden confundir
- **Lee lo que hace cada plugin** antes de habilitarlo
- **Algunos plugins necesitan configuración extra** como conectarse a servicios externos
</Aside>

## Añadiendo Nuevas Habilidades

<Steps>
1. **Explora los plugins disponibles**
   
   Mira la lista de habilidades que puedes dar a tu asistente.

2. **Lee la descripción**
   
   Entiende qué hace la habilidad y qué acceso necesita.

3. **Habilita el plugin**
   
   Actívalo para que la habilidad esté disponible para tu asistente.

4. **Configúralo**
   
   Proporciona cualquier información necesaria como detalles de conexión o preferencias.

5. **Prueba en el Área de Pruebas**
   
   Asegúrate de que tu asistente use la nueva habilidad correctamente antes de que los clientes la prueben.
</Steps>

## Configurando Habilidades

Después de habilitar un plugin, haz clic en configuración para configurarlo. Podrías necesitar conectarte a servicios externos, proporcionar información de cuenta o establecer reglas para cuándo tu asistente debe usar esta habilidad.

<Tabs>
  <TabItem label="Tipos de Configuración">
    | Tipo | Qué Necesitarás |
    |------|-----------------|
    | **Conexiones** | Info de login para servicios externos |
    | **Notificaciones** | Dónde enviar alertas cuando algo suceda |
    | **Disparadores** | Reglas para cuándo tu asistente debe usar esta habilidad |
    | **Permisos** | Qué información puede acceder el plugin |
  </TabItem>
  <TabItem label="Cuándo Usar Habilidades">
    | Disparador | Cuándo Tu Asistente Usa la Habilidad |
    |-----------|-------------------------------------|
    | **Auto-Detectar** | Cuando reconoce que un cliente quiere esto |
    | **Palabras Clave** | Cuando el cliente dice ciertas palabras |
    | **Siempre Disponible** | Tu asistente puede usarla cuando tenga sentido |
    | **Solo por Solicitud** | Solo cuando el cliente pide específicamente |
  </TabItem>
</Tabs>

<Aside type="tip" title="Consejos de Configuración">
- **Sigue las instrucciones de configuración** paso a paso
- **Prueba en el Área de Pruebas** antes de dejar que los clientes usen nuevas habilidades
- **Algunas habilidades necesitan cuentas** con otros servicios (como Calendly para programación)
</Aside>

## A Qué Pueden Acceder los Plugins

<Aside type="caution" title="Verifica Antes de Habilitar">
Algunos plugins necesitan acceso a información de clientes o servicios externos. Revisa qué necesita cada plugin antes de activarlo, y asegúrate de estar cómodo con lo que puede ver y hacer.
</Aside>

## Construyendo Habilidades Personalizadas <Badge text="Avanzado" variant="default" size="small" />

Para aquellos que quieren construir sus propios plugins:

```javascript
// Ejemplo: Habilidad personalizada
{
  "name": "verificar_inventario",
  "description": "Verificar cuántos items hay en stock",
  "parameters": {
    "product_id": "string"
  },
  "endpoint": "https://tu-api.com/inventario"
}
```

<Aside type="note" title="Recursos para Desarrolladores">
Consulta la [Referencia de API](/es/api/overview/) para documentación completa sobre cómo construir plugins personalizados.
</Aside>

## Consejos para el Éxito

<Steps>
1. **Comienza con lo que necesitas**
   
   Más plugins = más complejidad. Empieza simple.

2. **Prueba todo**
   
   Asegúrate de que las habilidades funcionen correctamente en todas las situaciones.

3. **Establece reglas claras**
   
   Ayuda a tu asistente a saber exactamente cuándo usar cada habilidad.

4. **Monitorea los resultados**
   
   Usa Analíticas para ver qué habilidades son más valiosas.
</Steps>
