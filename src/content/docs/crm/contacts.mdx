## Managing Contacts

<Aside type="tip" title="Contact Management">
- **Use tags** to organize contacts by type, source, or status
- **Review duplicates** regularly to keep data clean
- **Custom fields** help you track information specific to your business
</Aside>

### Create a Contact

<Steps>
1. **Click + New Contact**
   
   Access the contact creation form from the Contacts page.

2. **Fill in details**
   
   Enter basic information like name, email, and phone.

3. **Add tags**
   
   Apply relevant tags for organization (optional).

4. **Click Save**
   
   The contact is now in your CRM and ready for engagement.
</Steps>

## Import Contacts

<Steps>
1. **Navigate to Import**
   
   Go to **Contacts** ‚Üí **Import**

2. **Upload CSV file**
   
   Select your prepared CSV file.

3. **Map columns**
   
   Match CSV columns to CRM contact fields.

4. **Handle duplicates**
   
   Choose how to manage existing contacts.

5. **Complete import**
   
   Review and confirm the import.
</Steps>

<Tabs>
  <TabItem label="CSV Format">
    ```csv
    name,email,phone,company
    John Doe,john@example.com,+1234567890,Acme Inc
    Jane Smith,jane@example.com,+0987654321,Tech Corp
    ```
  </TabItem>
  <TabItem label="Import Options">
    | Option | Behavior |
    |--------|----------|
    | **Skip duplicates** | Don't import if email exists |
    | **Update duplicates** | Merge with existing record |
    | **Create duplicates** | Allow multiple records |
  </TabItem>
</Tabs>

## Export Contacts

<Steps>
1. **Filter contacts**
   
   Apply filters to select specific contacts (optional).

2. **Click Export**
   
   Start the export process.

3. **Select fields**
   
   Choose which fields to include in the export.

4. **Download CSV**
   
   Save the file to your computer.
</Steps>
