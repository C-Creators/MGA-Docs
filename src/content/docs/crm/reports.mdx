## Building Reports

<Tabs>
  <TabItem label="Quick Reports">
    **Pre-built reports ready to use**
    
    <Steps>
    1. **Go to Reports**
       
       Navigate to the Reports section from the sidebar.
    
    2. **Select report type**
       
       Choose from Contact, Conversation, Ticket, Campaign, or Lead reports.
    
    3. **Choose date range**
       
       Set the time period you want to analyze.
    
    4. **View results**
       
       See your data visualized instantly.
    </Steps>
  </TabItem>
  <TabItem label="Custom Reports">
    **Build your own reports** <Badge text="Advanced" variant="note" size="small" />
    
    <Steps>
    1. **Click + New Report**
       
       Start a custom report configuration.
    
    2. **Select data source**
       
       Choose which data to report on.
    
    3. **Choose metrics**
       
       Select what to measure (counts, averages, rates).
    
    4. **Add dimensions**
       
       Define how to group the data.
    
    5. **Apply filters**
       
       Narrow down to specific criteria.
    
    6. **Save report**
       
       Name and save for future access.
    </Steps>
  </TabItem>
</Tabs>

<Aside type="tip" title="Report Building">
- **Start with pre-built reports** and customize from there
- **Save frequently used reports** for quick access
- **Share reports** with team members for collaboration
</Aside>

## Report Components

<Tabs>
  <TabItem label="Metrics">
    **What to measure**
    
    | Metric Type | Examples |
    |-------------|----------|
    | **Count** | Total contacts, messages |
    | **Sum** | Total revenue, hours |
    | **Average** | Avg. response time |
    | **Percentage** | Conversion rate |
    | **Rate** | Messages per hour |
  </TabItem>
  <TabItem label="Dimensions">
    **How to group data**
    
    | Dimension | Use Case |
    |-----------|----------|
    | **Date/Time** | Trends over time |
    | **Category** | By type or status |
    | **User/Assignee** | Team performance |
    | **Segment** | Customer groups |
    | **Platform** | Channel comparison |
  </TabItem>
  <TabItem label="Filters">
    **What to include**
    
    | Filter Type | Example |
    |-------------|---------|
    | **Date range** | Last 30 days |
    | **Specific values** | Status = "Open" |
    | **Conditions** | Value > $1,000 |
    | **Segments** | VIP customers only |
  </TabItem>
</Tabs>
