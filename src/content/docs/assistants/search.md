# Search

Quickly find anything in your MyGPTAssistants workspace.

## Global Search

Access global search with `Cmd/Ctrl + K` or click the search icon.

### What You Can Search

- **Bots** - Find assistants by name
- **Conversations** - Search message content
- **Contacts** - Find people in your CRM
- **Data Sources** - Locate knowledge content
- **Settings** - Jump to configuration pages

### Search Syntax

**Basic search:**
```
customer support
```

**Exact phrase:**
```
"order status"
```

**Filter by type:**
```
type:conversation shipping issue
```

**Date range:**
```
after:2024-01-01 refund
```

## Bot Search

Within a bot, search your knowledge base:

1. Go to **Data Sources**
2. Click **Search**
3. Enter your query
4. See matching content chunks

This helps you:
- Verify content is indexed
- Find specific information
- Debug retrieval issues

## Conversation Search

Find specific conversations:

### Filters

- **Date range** - When conversation occurred
- **Bot** - Which assistant
- **Status** - Active, resolved, etc.
- **Contact** - Specific person
- **Contains** - Message content

### Search Tips

- Use specific terms for better results
- Combine filters to narrow down
- Export results for analysis

## Contact Search

Find contacts in your CRM:

- Search by name, email, phone
- Filter by tags, segments
- Sort by recent activity
