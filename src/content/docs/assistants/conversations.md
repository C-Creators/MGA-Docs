# Conversations

Access and manage all conversations happening with your AI assistants.

## Conversation List

The conversations view shows:

- **All conversations** across your bots
- **Status** - Active, resolved, escalated
- **Contact info** - Name, email (if captured)
- **Last message** - Preview of recent activity
- **Timestamp** - When conversation occurred

## Filtering & Search

### Filters

- **Bot** - Filter by specific assistant
- **Status** - Active, resolved, escalated, abandoned
- **Channel** - Web widget, WhatsApp, Instagram
- **Date range** - Custom time period
- **Tags** - Applied conversation tags

### Search

Search conversations by:
- Contact name or email
- Message content
- Conversation ID

## Conversation Detail

Click a conversation to view:

### Chat History

Full message history including:
- User messages
- Bot responses
- System events (handoff, tag applied)
- Timestamps

### Contact Information

Linked contact details:
- Name, email, phone
- Previous conversations
- CRM data (notes, tags)

### Actions

Available actions:
- **Tag** - Apply labels for organization
- **Assign** - Route to team member
- **Escalate** - Hand off to human agent
- **Resolve** - Mark as complete
- **Export** - Download transcript

## Live Chat / Handoff

When a conversation is escalated:

1. Agent receives notification
2. Agent can view full history
3. Agent takes over conversation
4. Bot can be re-enabled when done

### Handoff Settings

Configure in bot settings:
- Auto-escalation triggers
- Agent availability
- Queue management
- Notification preferences

## Bulk Actions

Select multiple conversations to:
- Apply tags
- Export transcripts
- Mark as resolved
- Delete (with confirmation)
