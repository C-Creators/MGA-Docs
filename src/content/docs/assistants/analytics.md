# Analytics

Understand how your AI assistant is performing with comprehensive analytics.

## Dashboard Overview

The analytics dashboard provides:

- **Conversation Metrics** - Volume, duration, resolution
- **User Engagement** - Active users, return rate
- **Bot Performance** - Response accuracy, handoff rate
- **Business Impact** - Leads generated, appointments booked

## Key Metrics

### Conversation Metrics

| Metric | Description |
|--------|-------------|
| **Total Conversations** | Number of chat sessions |
| **Messages per Conversation** | Average engagement depth |
| **Average Duration** | How long conversations last |
| **Resolution Rate** | Conversations resolved without human |

### Performance Metrics

| Metric | Description |
|--------|-------------|
| **Response Time** | Average time to first response |
| **Handoff Rate** | % escalated to human agents |
| **User Satisfaction** | Rating from feedback (if enabled) |
| **Fallback Rate** | % of questions bot couldn't answer |

### Business Metrics

| Metric | Description |
|--------|-------------|
| **Leads Captured** | New contacts created |
| **Appointments Booked** | Successful bookings |
| **Tickets Created** | Support tickets generated |
| **Conversion Rate** | Visitors → engaged users |

## Time Ranges

View data for:
- Last 24 hours
- Last 7 days
- Last 30 days
- Custom date range

## Filtering

Filter analytics by:
- **Bot** - Specific assistant
- **Channel** - Web, WhatsApp, Instagram
- **Tag** - Conversation tags
- **Outcome** - Resolved, escalated, abandoned

## Reports

### Scheduled Reports

Set up automated reports:
1. Go to **Analytics** → **Reports**
2. Click **+ New Report**
3. Select metrics and frequency
4. Add recipients

### Export Data

Export raw data for analysis:
- CSV format
- JSON format
- API access

## Insights

AI-powered insights highlight:
- Trending topics
- Common failure points
- Peak usage times
- Improvement opportunities
